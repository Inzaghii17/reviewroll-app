# ReviewRoll Flask API (Migration Skeleton)

This folder contains the migration skeleton for the future Flask API.

## Run

1. Create and activate virtual environment.
2. Install requirements:

```bash
pip install -r requirements.txt
```

3. Copy env:

```bash
copy .env.example .env
```

4. Start app:

```bash
python run.py
```

Health endpoint:
- GET http://localhost:5000/api/health
